# Fresh editor (2026-01)

I don't like to use VIM or nano. They are so different from my usual VSCode typing experience.  
Now there is this new Fresh Editor that is very similar to VSCode, but it runs inside the bash terminal.  

<https://github.com/sinelaw/fresh>

## Installation for Debian

Download latest release from GitHub:  
<https://github.com/sinelaw/fresh/releases>  
<https://github.com/sinelaw/fresh/releases/download/v0.1.75/fresh-editor_0.1.75-1_amd64.deb>  

Install from local deb file:

```bash
sudo dpkg -i fresh-editor_*.deb
```
